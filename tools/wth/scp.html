  • In the 1960s! That’s when the idea for a global network started. The U.S. Department of Defense made ARPANET. This was a big step because it used packet switching to send data.
  • Then came the 1970s and 1980s! The Transmission Control Protocol/Internet Protocol (TCP/IP) appeared. It helped different networks connect & communicate better. This was key for growing the Internet.
  • By the 1990s, everyday people could use it too! Commercial Internet Service Providers (ISPs) showed up. Plus, there was this cool thing called the Domain Name System (DNS). It made finding things online much easier!
  • In 1989, Tim Berners-Lee invented the World Wide Web while at CERN—how cool is that? He wanted to create a way to manage & share info using hypertext.
  • Fast forward to 1991! We saw the first website go live—woohoo! This was the start of something big.
  • Then in 1993, along came Mosaic. This web browser made using the WWW super easy & fun for everyone!
  • From then till now—Wow! The WWW exploded in growth! It changed from plain pages to interactive ones that are rich with all sorts of media.

  1. Crawling: Search engines send Tiny programs called crawlers or spiders to check out web pages all over the Internet. They gather data about what's on those pages.
  2. Indexing: The data collected gets stored in a huge database called an index. This makes it quick for search engines to find relevant info when someone searches.
  3. Ranking: When you type something in, the search engine looks at its index to figure out which results are the best fit. Things like keywords and page quality matter here!
  4. Retrieval: Finally, the search engine gives you a list of sorted web pages with little previews of what each page is about.
